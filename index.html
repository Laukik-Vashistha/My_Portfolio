Study Notion

StudyNotion is an innovative EdTech platform focused on improving education in rural areas by enabling instructors to easily upload and manage courses, providing students with effortless access to learning materials, and equipping administrators with comprehensive tools to oversee the system effectively. The platform utilizes Cloudinary for scalable media storage, JWT for secure authorization, and NodeMailer for OTP-based user authentication. With a frontend designed using Tailwind CSS and developed with React and Redux Toolkit, StudyNotion offers a responsive and user-friendly experience. The Node.js backend efficiently manages authentication and course operations, while MongoDB provides a flexible and scalable database solution. Razorpay integration ensures smooth and secure payment processing, making quality education more accessible to underserved communities.

MailFlow

This project is a bulk mailer application built using the MERN stack (MongoDB, Express, React, Node.js) and the SendGrid API. It enables users to send customized emails to groups of people, with MongoDB storing user data, mail groups, and templates. The backend is powered by Express and Node.js, handling HTTP requests and running the server-side code, while the frontend is developed with React and styled using Bootstrap. Axios is employed for making API requests to the backend. The application is designed to be customizable and extendable, serving as a valuable resource for learning or building similar bulk mailing projects.
